Few of my projects!

2-D Image to 3D Conversion Model

This project focuses on designing a scalable database to manage large volumes of 2-D image data and their corresponding 3D models. Key features include creating efficient ETL pipelines for data processing and optimizing queries for fast retrieval of image data.

Expense Manager

Developed a backend system using Flask to manage users' financial data, incorporating third-party APIs for currency conversion and financial data synchronization. The application also features an intuitive user interface for easy management of expenses.

Face Recognition Attendance System

This project implements a real-time face recognition system using OpenCV to accurately track attendance. The backend, built with Flask, integrates with Google Sheets for real-time attendance updates and features a user-friendly interface for both administrators and users.

Cryptocurrency Chart Viewer

This project involves building a web application using Node.js and the Binance API to display real-time cryptocurrency charts. It also allows users to add coins to a wishlist for quick access and monitoring of their favorite cryptocurrencies.

Sudoku Solver

Developed a Sudoku solver application using the brute force algorithm in Python. The user interface, built with Tkinter, allows users to input puzzles and get instant solutions. The backend logic utilizes backtracking to solve Sudoku puzzles efficiently.

Pollution Detection System

Built an IoT-based pollution detection system to monitor air quality in real-time. The system uses sensors to detect various pollutants and displays the data via a user-friendly interface, helping raise awareness about environmental conditions.

Simple Chatroom

Developed a real-time chatroom application using Flask and Socket.IO. It allows users to create or join rooms and chat with others in real-time. The app features message persistence and user session management, enhancing the user experience.
